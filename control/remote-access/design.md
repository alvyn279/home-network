# Remote Access - WireGuard VPN

## Overview
Secure remote access to home network via WireGuard server on TP-Link Archer AX23 router.

## Architecture
```
Remote Device → Internet → Telus Hub → Archer Router (WireGuard) → Home Network
                          192.168.1.1  192.168.1.119:41194    192.168.0.x
```

## Network Configuration
- **Telus Hub**: 192.168.1.1 (ISP router/modem)
- **Archer Router WAN**: 192.168.1.119 (static IP lease)
- **Archer Router LAN**: 192.168.0.1 (WireGuard server)
- **HP EliteDesk**: 192.168.0.68
- **VPN Client Pool**: 10.0.0.0/24

## Setup

### Port Forwarding (Telus Hub)
```
External Port: 41194 UDP
Internal IP: 192.168.1.119
Internal Port: 41194 UDP
```

### Router Configuration
1. **Access router**: http://192.168.0.1 → Advanced → VPN Server → WireGuard
2. **Enable WireGuard**: Port 41194, client pool (10.0.0.0/24)
3. **Create client account**: Add username, generate keys
4. **Download config** for client devices

### Client Setup
1. **Install WireGuard** (Mac App Store)
2. **Import config** from router
3. **Connect and test**

## Configuration Examples

### Router Settings
```
Listen Port: 41194
Client IP Pool: 10.0.0.0/24
DNS Server: 192.168.0.1
```

### Client Config
```ini
[Interface]
PrivateKey = [Generated by router]
Address = 10.0.0.2/32
DNS = 192.168.0.1

[Peer]
PublicKey = [Router public key]
Endpoint = home.yourdomain.com:41194
AllowedIPs = 192.168.0.0/24
PersistentKeepalive = 25
```

## Usage
```bash
# Connect WireGuard, then:
ping 192.168.0.1   # Router
ssh alvynle@192.168.0.68  # HP EliteDesk
curl http://192.168.0.68:3000  # Future Grafana
```

## Benefits
- **Fast**: Kernel-level implementation
- **Simple**: Router-managed client accounts
- **Secure**: Modern cryptography (ChaCha20, Curve25519)
- **Split tunneling**: Only home traffic uses VPN
- **Key generation**: Router generates private key, client derives public key using Curve25519 elliptic curve
